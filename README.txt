## Biblioteca UGB

- Projeto WEB, simulando a Biblioteca e seu acervo. (Contando com uma pequena quantia de livros para exemplo exclusivo da funcionalidade do sistema como a filtragem e avaliação.)

## Tecnologia 

* HTML
* CSS
* JavaScript
* Django

## Serviços

* GitHub

## Funcionalidades

- Login funcional (Matrícula de exemplo: 12345), (Senha de exemplo: 123) 
- logout no canto superior direito caso queira rever a página de login
- Página inicial com todos os livros disponíveis em um carrossel
- Acervo com os livros e opções de filtragem como:

* Autores
* Ano de postagem do livro

- Dark Mode
- Barra de busca
- Caso clique no livro, verá um botão de reservar
- Na página de detalhes do livro, poderá avalia-lo com 5 estrelas.
